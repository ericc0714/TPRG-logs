# 📘 第 8 回 GM 回合總結（正式版）
副標題：核心共鳴、命運交錯的森林之夜

## （一）行動判定區塊
### 【墨嵐】行動判定
行動內容： 維持環境監視 → 將魔力砲能量轉換為大地元素操作 → 嘗試構築高機動魔力機車。
判定：部分成功（d6 = 4）
* 能成功塑形出「大地元素骨架」並以魔力融合。
* 底盤穩定、能移動，但還稱不上“機車”，較像 簡易魔動滑行器。
* 風系降低風阻術式可運作，但效率中等。
* 若強行加速，可能失衡或解體（需後續調校）。

### 【每天都要吃一根香蕉】行動判定
行動內容： 穿越森林 → 察覺容器外溢能量被壓制 → 與玲交會 → 展現善意並提出詢問。
判定：成功（d6 = 6）
* 完全避開追兵與魔物。
* 容器的能量穩定效果顯示 你靠近“核心碎片”持有者。
* 與玲會面順利，對話自然產生進展。
* 你成功向玲提出關鍵問題，並讓她願意同行。

### 【玲】行動判定
行動內容： 遭遇香蕉 → 感到神秘熟悉感 → 社交混亂 → 提供藥水 → 決定同行。
判定：成功（d6 = 5）
* 共鳴造成的情緒混亂「合理但不失控」。
* 成功給予香蕉藥水，使他恢復些許體力。
* 順利決定與香蕉同行，沒有遭遇危險。
* 共鳴壓制容器能量 → 造成局部“安全區域”。

## （二）小說形式敘事
森林夜色深沉，濃霧沿著地表蠕動，宛如某種脈動般回應著地底的震鳴。

### 📍【香蕉與玲的相遇】
香蕉推開被夜風吹得微微顫動的灌木時，那股混沌不明的外溢能量突然消失了。
彷彿有人伸手撫平一片躁動的湖面。
他抬起頭，便看見那位紫瞳少女。
玲靜靜站在林間的縫隙裡，像是被世界遺落的影子。眼神裡有疲倦，也有不安。但當容器靠近她時，那股能量竟像被溫柔抱住般，徹底平穩。
香蕉先開口了。
「……這顆東西，應該是屬於妳的吧？」
說著，他抬起沉重的金屬容器，像是展示一件危險卻又無辜的寶物。
「剛剛它像要把整片森林炸掉。但到了妳面前——」
他停頓了一秒。
「像個睡著的孩子。」
玲怔怔地望著他——不，是望著容器。 胸口有一股奇妙的牽引感，彷彿身體深處有個遺失的東西正被喚醒。
臉頰突然發熱。
「我、我……不知道……」 她搖頭，不確定是對容器說、不確定是對他說， 甚至連自己在否認什麼都說不清楚。
但她仍伸出手，笨拙地遞上一小瓶藥水。
「你看起來……很虛弱。先喝這個吧。」
香蕉愣了一下，但還是接過，喝下。 草本香味散開，他胸口那股緊縮感終於稍稍舒緩。
少女重新抬頭時，眼神已不再只是戒備—— 而像是找到某種莫名能依靠的東西。
「我……可以和你一起走嗎？」
香蕉點頭。
「當然。」
容器在兩人之間微微脈動，像是回應這個選擇。

### 📍【墨嵐的高地】
在兩人相遇的同時，距離數百公尺外的高地上， 墨嵐正操縱魔力，讓大地流動。
岩塊像液態般重塑，組成框架。 魔力線交織，形成能量導流管。 細密的風系符文刻在外殼，降低摩擦。
一台 初代魔力機動裝置 在他腳下成形。
還不完美。 還需要穩定。 但足以在必要時—— 衝入戰場。
他抬頭。
遠方的樹林深處，有兩股小小的光點靠在一起。 玲與另一個陌生男子。
再遠一點，地底的震鳴愈來愈近。
「……事情，要加速了。」
他跨上尚未調整好的滑行器，魔力在輪軸下方嗡鳴。
而夜色中，古代兵器的低吼隱約回應。

## （三）場景／旁白（世界狀態）

### 📌 角色位置
* 玲 ＆ 香蕉 　→ 已正式同行，位於森林深處「核心能量共鳴區」。
* 墨嵐 　→ 位於高地，製造出試作魔力機車，可快速趕往現場。
* 追兵 　→ 因示警射擊仍不敢靠近，但正在召集支援。

### 📌 古代兵器狀態（主線核心）
* 目前處於 半甦醒階段。
* 地底震動周期縮短。
* 正以「微弱吸力」嘗試拉回自己的核心碎片（玲）。
* 同時也對啟動器（香蕉的容器）產生遠距引動。
* 若三人距離再靠近 → 會進入同步共鳴（高危險事件）。

### 📌 容器（啟動核心）狀態
* 靠近玲時： 　→ 100% 安定（進入鎮定模式）。
* 離開玲後之後： 　→ 逐漸重新活化（危險上升）。
* 目前正在： 　→ 反向牽引玲體內的碎片能量。
* 若強行啟動可能造成： 　→ 玲體內碎片暴走 　→ 兵器提前甦醒 　→ 魔物暴走加劇
