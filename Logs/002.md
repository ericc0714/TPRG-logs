# 📘 第 2 回（Round 2）GM 總結

## （一）行動判定區塊
### 【每天都要吃一根香蕉】
行動：察覺能量壓制 → 撤退 → 尋找能量屏蔽斗篷。 判定：成功 你成功判斷「虛弱是外界壓制造成」而非身體問題，並在壓制加劇前撤離。 你順利回到廢墟中的雜貨商據點範圍（尚未抵達商人面前，但非常接近）。
外界能量影響 → 下降為『微弱壓制』（仍可正常行動）。

### 【墨嵐】
行動：嘗試編寫魔法術式、模擬能量、奪取設施掌控權。 判定：部分成功 你能讓自身能量「短暫共鳴」但無法完全重現那種混合能量。 然而你的行動觸發了研究所內部的殘留設備反應。
✔ 成果：
* 你捕捉到該能量 不是自然魔力，也不是科技能量，而是： 『被強制轉換的魔力碎片』
* 你的干涉成功讓研究所的一扇 地下通道門短暫解鎖
✔ 但也有後果：
* 你吸引了位於研究所深處某「感測裝置」的注意
* 研究所內部開始傳出「低頻震動」
你沒有受到傷害。

### 【玲】
行動：前往能量源 → 環繞觀察 → 遭能量排斥 → 昏迷。 （此為 Round 2 行動，已確認沿用原內容） 判定：失衡／昏迷（非傷害） 你靠近能量源時遭到強烈排斥，這是你的體質特別敏銳所造成的「魔力過載」。 目前狀態：昏迷、呼吸穩定、可被發現／救援／自然醒來（需要時間）
你的昏迷位置： 倒塌高樓外圍陰影處（靠近能量源 30～40 公尺）

## （二）小說式敘事
破碎的舊城廢墟間，能量殘波像風一樣在空氣中擴散，忽強忽弱，彷彿有某種存在正在深處呼吸。

### 每天都要吃一根香蕉
在快被壓制到跪下之前，你及時轉身撤離。 一旦遠離北方核心區，那股沉重的壓力漸漸退去，身體重新恢復輕盈。
你沿著熟悉的巷道返回廢墟市場區，破布帳棚間冒著微弱的燈火。 那位賣各種奇怪雜物的老人，正坐在堆滿零件和破斗篷的攤位後面。
你知道—— 他手上很可能真的有你需要的「阻能斗篷」。

### 墨嵐
在研究所入口附近，你閉上眼，伸手向身前虛空描繪術式。 能量在你指尖聚集，空氣中浮起淡淡藍光，你以自己能量的節奏開始「模擬與干涉」。
──這時，牆壁中傳來回應。 像是某種被封存的脈動，被你的能量喚醒。
下一刻，一扇藏在瓦礫下的地下鐵門「喀」地鬆開機關，向下露出一條漆黑走廊。 裡面吹出冰冷空氣，伴隨著微弱、頻率極低的震動。
你察覺： 裡面某種「探測器」開始活動了。

### 玲
小小的身影倒在廢墟陰影中，呼吸微弱但規律。 周遭空氣因殘波扭曲而微微顫動，卻不再像之前那樣猛烈攻擊你。
你的手指微微顫動。 你聽不見聲音，但在半夢半醒間，你感受到某種熟悉而遙遠的魔力氣息。 像是—— 煉金術士曾經給你展示過的「魔力賦予模型」的一片碎影。
或許，這能量與他留下的研究……並非無關。
陰影下的你，正等待誰會成為第一個注意到你的人。

## （三）下一場景／回合 3 起始狀態
### 📍 各角色位置（非常重要）
* 每天都要吃一根香蕉 　→ 回到廢墟市場、即將接觸雜貨商 　→ 有機會取得「阻能斗篷」等裝備
* 墨嵐 　→ 研究所地表層 　→ 已打開「地下門」 　→ 內部啟動感測裝置（潛在事件源）
* 玲 　→ 能量源外圍、昏迷 　→ 隨時可能被香蕉或墨嵐發現（若兩人往北）
### 📌 重要世界同步
* 能量源正在活化
* 地下研究所似乎與能量泄漏直接相關
* 三人距離開始縮短，極可能在第 3～4 回正式交會
