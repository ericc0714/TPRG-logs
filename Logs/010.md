# ⭐⭐⭐ 第 10 回・GM 總結（正式判定版） ⭐⭐⭐

## （一）行動判定區塊：以六段式成功機制重審

### 🟣 1. 玲（itou_tsuki）行動判定 →《成功》
行動內容：
* 為香蕉準備衣物、藥水、裝備
* 整理工房、挑選能給的道具
* 與香蕉互動、引導補給流程
判定理由：
* 這些都是她能力範圍內可完美做到的事情
* 工房是玲的主場 → 加值
* 無外力干擾
* 唯一風險點（容器與她共鳴）被避免了，未觸碰啟動器
結果：
✔ 完整達成行動，沒有負面影響 ✔ 香蕉獲得裝備與補給 ✔ 劇情互動順利推進
—

### 🟡 2. 香蕉（ㄇㄆ）行動判定 →《部分成功》
行動內容：
* 接受防護衣、補給並挑選急救箱
* 與玲同行離開工房
* 下定決心「帶玲前往組織據點」
* 行經大地裂縫區域
判定理由：
* 補給、換裝 → 無難度，100%成功
* 但「打算帶玲前往組織」屬於高風險動作旗標，具有三項隱性難點：
    1. 組織是否真的能讓他帶陌生人回去？（社會限制）
    2. 裂縫區域極度不安定，靠近後容器開始出現微反應（環境風險）
    3. 香蕉本人對事態的理解有限（資訊風險）
→ 因此此目標不可能「完全順利」。 → 我給予《部分成功》是因為 你們成功踏上前往組織的路，但會遇到環境阻礙。
結果：
✔ 香蕉成功帶玲往目標方向前進 ⚠ 但靠近裂縫後——
* 容器開始有微弱震動（第一個負面徵兆）
* 地底古代兵器加速甦醒（香蕉不知情）
* 組織線在未來可能不順利
→ 這些屬於《部分成功》下的代價。
—

### 🔵 3. 墨嵐（言 張）行動判定 →《部分成功》
行動內容：
* 遠距操作複合觀測系統
* 解析整個區域的魔力流動
* 嘗試提升魔力理解度
判定理由：
這次是你三人行動中最難的一項，理由：
* 觀測範圍巨大（裂縫 + 魔物群 + 追兵 + 古代兵器 + 核心碎片）
* 魔力結構極度複雜
* 能量干擾劇烈（古代兵器甦醒中）
* 墨嵐雖然有天賦，但目前解析能力尚不足以「看透全域」
因此—— 你「成功解析部分魔力結構」，但「無法全數理解」。
結果：
✔ 魔力解析 → 部分成功 ✔ 魔力理解度提升（技能向成長） ⚠ 無法取得完整魔力地圖 ⚠ 若繼續深入解析，可能反噬（未來風險）
→ 標準的《部分成功》：主要達成，但仍有限制。

## （二）小說敘事版本（依照判定差異調整）
夕陽餘光灑落在破碎的煉金工房裡，牆壁起伏著金紅色的光。 玲在堆滿藥材與器具的案桌間穿梭，如同在廢墟中發光的小動物。
香蕉看著她忙碌的背影，心裡第一次浮起一種奇妙的感覺： 這孩子到底生活在什麼樣的地方？ 怎麼能隨手拿出這麼多珍貴的東西？
煉金工房裡的補給一件件堆進他懷裡—— 藥瓶、裝備、最後還有一件玲親手製的能量防護衣。
「這件真的很耐用喔，穿上比較安全！」 玲笑著說。
香蕉一度想拒絕，卻被她認真又略帶固執的神情鎖住了。 於是他收下了。
只是在心底深處…… 那份對玲的「熟悉感」依舊像細線般牽動著他。
兩人離開村落，往組織據點的方向前進。 但剛踏入接近大地裂縫的地帶時——
嗡……
香蕉腰間的容器輕輕震動了。 如同心臟的跳動。 又如同某個沉睡的怪物翻了個身。
香蕉愣了一瞬，但玲似乎沒察覺異狀。 裂縫下方的能量，正逐漸甦醒。
遠處的荒原上， 墨嵐站在高地，以術式展開的觀測視界猶如千面鏡片。
他看到魔力像潮汐般湧動： 追兵、魔獸群、古代兵器的能量框架、核心碎片的回聲……
其中有一條線最刺眼—— 來自玲的純白光脈。
解析並不完全，許多區域仍混沌不明。 但他已明白一件事：
如果古代兵器完全甦醒，三人現今的力量根本不夠。
而此刻—— 龜裂的大地深處傳來低沉的迴響。
彷彿某個巨大的存在正在呼吸。
